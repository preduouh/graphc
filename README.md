##  Aplicacao online ##



##  Como  rodar  ##

1. Abra o terminal na raiz do projeto e instale as dependências:
```bash
npm install
```
ou
```bash
yarn
```

2. Rode o projeto:
```bash
npm start
```
ou

```bash
yarn start
```
