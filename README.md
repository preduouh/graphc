##  Online Application ##



##  Como  rodar  ##

1. Open the terminal at the root of the project and install the dependencies:
```bash
npm install
```
or
```bash
yarn
```

2. Run the project:
```bash
npm start
```
or
```bash
yarn start
```

## Technologies and Libraries Used

* ReactJS
* ThreeJS
* Styled Components
* React Icons
